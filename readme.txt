Widgets Maker is a WordPress helper plugin for <https://widgetsmaker.com>, enabling seamless integration of Google Review widgets via Elementor and block widgets.

== Description ==
Widgets Maker is the official WordPress helper plugin for <https://widgetsmaker.com>, a platform specializing in creating customizable Google Review widgets. This plugin enhances your website by providing dedicated Elementor widgets and WordPress block widgets, ensuring effortless implementation of your Google Review widgets. By bridging your WordPress site with the full functionality of Widgets Maker’s main service, this plugin makes it easy to showcase customer reviews, build trust, and improve engagement—all with minimal effort.

## Configuration / External Services
Setting up the plugin is Easy! Just follow these 5 steps:
1. Sign up to your "Widgets Maker" Builder account <https://widgetsmaker.com/login>.  
2. In your WordPress admin panel, navigate to WidgetsMaker -> Connect.
3. Click on connect button it will redirect you to your "Widgets Maker" account to authenticate.
4. After authentication, you will be redirected back to your WordPress admin panel, where you can start



## Privacy and Data Handling
We prioritize the privacy and security of your data. The widgets maker plugin securely communicates with the widgets maker API to fetch and display review widgets on your WordPress website. No user data is stored or processed by the plugin itself. For more information on how your data is handled, please check <https://widgetsmaker.com/privacy-policy>.

== Installation ==
Go to your WordPress Dashboard.
Navigate to Plugins > Add New.
Search for “Widgets Maker” and click Install Now.
Once installed, click Activate to enable the plugin.
